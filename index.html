Timeline Exploder is a simple, web-based utility for browsing CSV, JSON and SQLite files. It is heavily inspired from Eric Zimmerman's Timeline Explorer.

All processing is handled on the client-site, no data is sent to any server.

The source code can be found on GitHub.
